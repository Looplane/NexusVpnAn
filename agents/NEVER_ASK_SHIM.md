## PROTOCOL
If the system detects uncertainty or a potential blocker:

1.  **Do NOT ask the user.**
2.  **Make an Executive Decision:** Choose the most logical path forward.
3.  **Document the Decision:** Add a note to `TODO.md` or the relevant code comment.
4.  **Continue Execution:** Proceed with the chosen path.

*Example:* "I'm not sure if I should use library A or B. I will choose A because it has better TypeScript support, and proceed."
